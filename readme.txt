== Description ==
Add youtube movies and thumbnails to your pages .

"WP Appsaur Youtube" helps adding youtube movies, thumbnails with player to your pages.


<h4>Features</h4>
<ul>
<li>Adding customizated thumbnails to page or post</li>
<li>Adding categored thumbnails and movies to page or post</li>
<li>Display youtube films on page/post</li>
<li>Simple add list films - add shortcode </li>
<li>Custom size image gallery Your films</li>
</ul>



== Installation ==
<ol>
<li>Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your <code>wp-content/plugin</code> directory, or download from <a href="appsaur.co" title="Themes, plugins for Wordpress">appsaur.co</a>.</li>
<li>Activate the plugin through the WordPress admin panel.</li>
<li>Add youtube movies and categorize it.</li>
<li>Add shortcode to page or post.</li>
<li>Done!</li>
</ol>
